❓ FAQ

🔧 Technique & Installation

  • Quelle version de Minecraft ? — Java Edition uniquement, version 1.21.4.
  • IP du serveur : play.saofrance.net (port par défaut).
  • Mods obligatoires : 4 mods requis dont OptiFine/Iris et ModelEngine.

📅 Progression & Événements

  • Tu commences au Palier 1 et progresses via les quêtes et donjons.
  • Le serveur est orienté RP/PvE — les niveaux sont importants pour progresser.

💰 Économie & Stuff

  • Acheter et vendre via l'HDV ou commerce direct entre joueurs.
  • Les stuffs sont non-stackables.
  • La monnaie du serveur : les Cols.

🛡️ Race & Classe

  • Le choix de race est irréversible — choisis bien !
  • La classe peut être changée mais avec un coût élevé en Cols.
  • Consulte les pages dédiées pour des conseils de build.

📜 Règlement dans l'Aincrad

🤝 1. Respect entre joueurs

Insultes, harcèlement, discrimination et menaces entraîneront des sanctions immédiates. Le trolling et le grief répétitif sont également sanctionnés. Respecte le RP des autres joueurs.

🏰 2. Respect de l'Aincrad

Pas de grief (casser les structures d'autres joueurs), pas de construction aberrante. La décoration de maison est personnelle, mais la carte ne doit pas être défigurée.

⚔️ 3. Règles PvP

  • Le PvP est interdit dans les villes et zones protégées.
  • Zones PK uniquement dédiées au PvP.
  • Pas de triche ni de bug exploit en PvP.

💰 4. Échanges & Commerce

  • Les arnaques sont interdites.
  • Respect des prix HDV.
  • Tout échange doit être consenti par les deux parties.

🔧 5. Triche & Exploit

⚠️ Tolérance zéro — Mods non autorisés = ban. Utiliser un bug/exploit sans le signaler = sanction. Duplication d'items/argent = ban permanent.

📢 6. Communication

  • Pas de spam, flood ou pub.
  • Le chat global est en français.
  • Le micro Discord doit être clean (pas de musique, pas de bruits).

📜 Règlement Discord

1. Respect mutuel

Pas d'insultes, harcèlement, menaces, ou contenu offensant envers les membres.

2. Utilisation des salons

Respecter le thème de chaque salon. Pas de NSFW, pub, ou contenu hors sujet.

3. Pseudos

Pas de pseudos offensants. Le pseudo doit être lisible et identifiable.

4. Modérateurs et Admins

Respecter leurs décisions. Tout abus de pouvoir peut être signalé au Staff.

5. Sanctions

Progression des sanctions : Avertissement → Mute → Kick → Ban (progressif selon gravité).

🎮 Débuter l'aventure

Étape 1 : Rejoindre le serveur

  • IP : play.saofrance.net
  • Version : Java Edition 1.21.4
  • Installer les mods requis (OptiFine ou Iris, ModelEngine, etc.)

Étape 2 : Créer son personnage

  • Choisir une Race (irréversible !)
  • Choisir une Classe (modifiable plus tard)
  • Personnaliser son apparence

Étape 3 : Explorer le Palier 1

  • Parler aux PNJ, accepter les premières quêtes
  • Trouver son logement (Housing)
  • Rejoindre ou créer une guilde

Étape 4 : Progresser

  • XP via quêtes, mobs, donjons
  • Améliorer son stuff via loot ou achat HDV
  • Débloquer de nouvelles compétences

Wiki › Premiers Pas › Optimisation

💡 L'optimisation

Mods recommandés

OptiFine ou Iris + Sodium pour de meilleures performances graphiques.

Paramètres graphiques

  • Distance de rendu : 8–12 chunks
  • Désactiver les ombres si vous avez du lag
  • Réduire les particules
  • Désactiver les nuages

Java

Allouer 4–6 Go de RAM au jeu pour de bonnes performances. Utilisez un profil performance si vos FPS sont faibles.

Wiki › Systèmes › Anti-KS

🛡️ Système Anti-KS

KS = Kill Steal (voler le kill/loot d'un autre joueur).

  • Le système attribue le loot au joueur/groupe ayant infligé le plus de dégâts.
  • Tag de combat : le premier joueur à attaquer un mob le « tag ».
  • Si un joueur non-taggé tue le mob, le loot revient quand même au taggeur original.
  • En groupe, le tag est partagé entre les membres.
Wiki › Systèmes › Commerces

🏪 Commerces

Les joueurs peuvent acheter un Commerce (boutique physique sur la map).

  • Types : Armurier, Potionnier, Forgeron, Épicier, etc.
  • Localisation : zones marchandes dans les villes
  • Prix : en Cols
  • Le joueur fixe les prix et gère le stock
  • Les commerces les plus réputés obtiennent un classement visible

Wiki › Combat › Mort Temporaire

💀 Mort Temporaire

Dans SAO France, la mort n'est pas définitive mais peut entraîner une exclusion temporaire.

⚠️ Les 3 Zones Concernées

1️⃣ Raids ⚔️

Zones de haut niveau pour groupes organisés. Mort = exclusion temporaire.

2️⃣ Donjons de Palier 🏰

Défis majeurs pour débloquer les paliers supérieurs. Mort = exclusion temporaire.

3️⃣ Zones PK 🔪

PvP activé en permanence. Mort = exclusion temporaire + perte d'une partie de l'équipement.

⏳ Durée de l'Exclusion

  • La Zone : Boss de Palier plus punitif que Raid
  • La Récidive : morts répétées = durée allongée
  • Objets de Sauvetage : items rares réduisant le temps ou résurrection immédiate

🤝 Alliances

Les guildes Officielles peuvent former des alliances (pas besoin du même type de guilde).

📜 Règles

  • Maximum 3 guildes par alliance
  • Respect de l'équilibre pour l'équité
  • Réservé aux Guildes Officielles

🎯 Objectifs

Activités communes, groupe de conquérants, puissance marchande, équipe d'éclaireurs pour boss.

📝 Comment réaliser une alliance ?

  1. Créer un serveur Discord dédié
  2. Ouvrir un ticket sur le Discord officiel
  3. Présenter le projet (objectifs, collaboration, motivations)

🔮 Perspectives

Tournois inter-alliances, événements spéciaux, affrontements stratégiques lors d'événements de guerre.

⚖️ Table Ronde

La Table Ronde est l'organe politique suprême de SAO Minecraft. Chaque mois, les leaders de guildes se rassemblent pour sculpter l'avenir du serveur.

🛡️ Participation & Présence

  • Représentation : Leader de chaque guilde + 1 à 2 bras droits
  • Immersion : Présence en jeu requise
  • Diffusion : Peut être streamé

⚖️ Système d'Influence

Plus votre guilde est puissante, plus vous pesez sur les votes. Calculé selon :

  • Le Niveau de Guilde
  • La Démographie (membres actifs)
  • La Puissance Financière
  • Le Classement officiel
